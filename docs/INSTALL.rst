cisindia.multiviews Installation
--------------------------------

To install cisindia.multiviews using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``cisindia.multiviews`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        cisindia.multiviews

* Re-run buildout, e.g. with:

    $ ./bin/buildout
